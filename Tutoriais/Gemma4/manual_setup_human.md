# 👤 Gemma 4: Guia de Instalação Manual

1. **Instale o Ollama**: Baixe em [ollama.com](https://ollama.com).
2. **Baixe os Modelos**: No terminal, rode `ollama pull gemma4:31b`.
3. **Configure o Cursor**: Instale a extensão **Continue** e selecione o Ollama como provedor.
4. **Verificação**: Acesse `http://localhost:11434` no navegador.
