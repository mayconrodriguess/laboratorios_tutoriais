# 👤 Stitch MCP: Guia de Instalação Manual

1. Obtenha sua API Key em [stitch.withgoogle.com](https://stitch.withgoogle.com).
2. No Windows, configure seu `mcp.json` usando o caminho completo para o `npx.cmd`.
3. Use as capabilities `listProjects` e `getScreenCode` para acelerar seu desenvolvimento.
